## What this is
A minimal **AYA hub** landing + **Vault links** + copyable intake prompt.

## Deploy (GitHub Pages)
1. Create a new repo (public).
2. Upload all files from this folder to repo root.
3. GitHub → **Settings** → **Pages**
4. Source: **Deploy from a branch**
5. Branch: **main**, Folder: **/ (root)**
6. Save.

## Deploy (Netlify)
- Drag & drop the folder in Netlify (or connect repo).
- Build command: **none**
- Publish directory: **/**
